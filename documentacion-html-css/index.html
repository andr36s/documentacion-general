MANIPULACION DEL DOM

Métodos y propiedades que te permiten interactuar con los elementos HTML.
  • .childElementCount -> Obtiene el numero de elementos hijos de un nodo.
     ~ Ejemplo los hijos dentro de un div con <p>, <p> y <span> devuelve 3.
  • .children -> Devuelve los nodos hijos que son etiquetas HTML.
     ~ Puede devolverlos todos o uno en especifico .children[1]
  • .parentElement -> Se utiliza para acceder al elemento padre de un nodo, se puede usar varias veces de seguido.
  • .nextElementSibling -> Se utiliza para acceder al hermano nodo siguiente de un elemento.

Acceder al contenido o a las propiedades de un nodo.
  • .getElementById('id') -> Obtener un elemento por su ID.
  • .getElementsByTagName('tagName') -> Obtener un elemento por el nombre de la etiqueta.
  • .getElementsByClassName('className') -> Obtener un elemento por el nombre de la clase.
  • .querySelector('selector') -> Selecciona el primer elemento que coincida con el selector CSS.
     ~ Puede seleccionar combinaciones complejas de elementos, como .mi-clase > p:first-child.
     ~ Usa la misma sintaxis que CSS.
     ~ Ideal para seleccionar un único elemento rápidamente.
     ~ '>' Es clave para garantizar que tu selección sea específica y solo abarque los hijos directos del elemento padre. Para evitar seleccionar elementos a niveles más profundos del DOM.
     ~ Para etiqueta solo el nombre, # para id, . para class, etc.
  • .querySelectorAll('selector') -> Selecciona todos los elementos que coincidan con el selector CSS.
     ~ Trae una lista de nodos, no una colección de datos.
     ~ Puedes usar métodos como forEach, for...of, pero no métodos de array como map o filter.
     ~ La NodeList es estática, no se actualiza automáticamente.
     ~ Es útil para aplicar cambios o realizar operaciones en varios elementos a la vez.
     ~ Para etiqueta solo el nombre, # para id, . para class, etc.
